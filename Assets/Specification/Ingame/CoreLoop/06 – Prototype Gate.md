# 06 – Prototype Gate

본 문서는 코어 루프 프로토타입 완료 기준을 정의한다.

------

## Gate Checklist

- 고정 Tick(`2s`)이 씬 전환/일시정지/재개 후 유지된다.
- 상태 머신 순서가 강제된다.
- 플레이어 직접 조작 경로가 제거된다.
- 개입 대기열이 Tick 경계에서만 반영된다.
- 세션 종료 3조건이 동작한다.
- 종료 영속화 최소 세트가 저장된다.

## Pass Criteria

- 위 체크리스트 전 항목이 `True`일 때만 프로토타입 완료로 판정한다.

## Blockers

- 상태 전이 우회 가능성 발견
- 종료 조건 외 임의 종료
- 영속화 실패 후 종료 확정

위 항목 중 하나라도 발생하면 구현 완료 판정은 무효다.
