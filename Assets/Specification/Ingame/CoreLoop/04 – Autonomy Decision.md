# 04 – Autonomy Decision

본 문서는 캐릭터 자율 행동 결정 규칙을 정의한다.

------

## Decision Inputs

행동 결정은 다음 입력을 사용한다.

- 정책(지령)
- 성향 가중치
- 상태값(체력/스트레스/트라우마)
- 현재 루프 상태

## Selection Rule

- 매 Tick마다 후보 행동 집합을 계산한다.
- 후보 행동은 가중치 + 확률 요소로 선택한다.
- 확률 요소는 시드 기반 난수로 생성한다.

## Safety Rule

- 행동 선택 결과가 금지 상태 전이를 유발하면 해당 행동은 무효화한다.
- 무효화 시 대체 후보를 재평가한다.

## Replay Rule

- 디버그 모드에서 Tick별 결정 로그(입력, 후보, 선택 결과)를 재생성 가능해야 한다.
